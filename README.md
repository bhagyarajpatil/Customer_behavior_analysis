# Customer_behavior_analysis
Customer Behavior Analysis project using Python, SQL, and Power BI to analyze purchasing patterns and customer trends. Includes data cleaning, EDA, SQL queries, and interactive dashboards to visualize sales performance and customer insights.

Overview

This project analyzes customer purchasing behavior to identify buying patterns, sales trends, and customer insights. Using Python, SQL, and Power BI, the project transforms raw transaction data into meaningful insights through data cleaning, exploratory data analysis (EDA), and interactive dashboards.

Dataset:customer_shopping_behavior.csv

The dataset contains customer transaction information such as:

Customer ID

Product purchased

Order date

Quantity and price

Total purchase amount

This data is used to analyze customer purchasing trends and sales performance.

Tools & Technologies:

Python – Data cleaning and exploratory data analysis (Pandas, NumPy, Matplotlib, Seaborn)

SQL – Data querying and aggregation

Power BI – Dashboard creation and visualization

Jupyter Notebook – Data analysis environment

Steps Performed:

Data Cleaning

Handled missing values

Corrected data types

Removed duplicate records

Exploratory Data Analysis (EDA)

Analyzed sales distribution

Identified customer purchase trends

Visualized patterns using charts

Queried transaction data

Calculated sales metrics and aggregates

Dashboard Development

Visualized customer behavior and sales trends

Dashboard:
